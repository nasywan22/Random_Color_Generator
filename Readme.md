Random Color Generator

Description
The Random Color Generator is a simple web program that generates a random color with just a click of a button or a press of the spacebar. Users can easily copy the generated HEX code by clicking the "Copy HEX" button.

Features
Generate random colors with a single click or key press
Copy generated HEX code with ease
Getting Started
To use the Random Color Generator, simply open the program in a web browser and click the "Randomizer" button or press the spacebar to generate a random color. Click the "Copy HEX" button to copy the generated HEX code to your clipboard.

Technical Details
The Random Color Generator is built using native web technologies, with no dependencies on external frameworks or libraries.
